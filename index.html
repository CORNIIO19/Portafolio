Mis Proyectos Relevantes

Proyecto 1

Este proyecto de programación busca resolver la necesidad de una gestión eficiente de tareas y contenidos en entornos educativos. Se desarrolló una aplicación web utilizando XAMPP como entorno local, con PHP, MySQL, HTML/CSS y JavaScript, permitiendo registrar usuarios, asignar tareas y generar reportes de rendimiento. Mi rol se centró en el desarrollo del backend, diseñando la base de datos y programando las funciones principales para el manejo de usuarios y tareas. Gracias a este proyecto, fortalecí mis habilidades técnicas en bases de datos y lógica de servidor. Además, el trabajo en equipo y la comunicación fueron clave en el proceso, lo que me permitió mejorar significativamente mis habilidades blandas, como la colaboración, la organización y la adaptación a distintos enfoques de trabajo.
